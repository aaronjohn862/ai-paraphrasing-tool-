# 🤖 AI-Powered Paraphrasing Tool

An AI-based paraphrasing tool built using transformer models that rewrites text while preserving its original meaning, improving clarity, and ensuring grammatical correctness.

---

## 🚀 Features

- Transformer-based paraphrasing (T5 model)
- Grammar and spelling correction
- Evaluation metrics:
  - BLEU Score
  - ROUGE Score
  - Semantic Similarity
- Simple console-based interface

---

## 🧠 Technologies Used

- Python  
- Hugging Face Transformers  
- PyTorch  
- Sentence Transformers  
- NLTK  
- ROUGE Score  
- LanguageTool
