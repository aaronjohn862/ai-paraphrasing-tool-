# 🤖 AI-Powered Paraphrasing Tool

This project is an AI-based paraphrasing tool that rewrites input text while preserving its original meaning, improving clarity, and ensuring grammatical correctness.

---

## 🚀 Features

- Transformer-based paraphrasing using T5 model
- Grammar and spelling correction
- Evaluation metrics:
  - BLEU Score
  - ROUGE Score
  - Semantic Similarity
- Console-based input/output (no GUI)

---

## 🧠 Technologies Used

- Python
- Hugging Face Transformers
- PyTorch
- NLTK
- Sentence Transformers
- LanguageTool
